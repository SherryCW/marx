# Marx Perspective · 马克思视角

> A bilingual Codex Skill that distills Karl Marx’s methods of analysis into a runnable thinking advisor.  
> 一个双语 Codex Skill：将卡尔·马克思的分析方法提炼为可运行的思维顾问。

## What it contains · 内容

The skill distills six core mental models:

1. **Historical conditioning** — give every category a “birth certificate”.  
   **历史条件化**——给每个范畴查“出生证”。
2. **Making relations visible** — move from things, prices, and legal forms to social relations.  
   **关系显形**——从物、价格和法律形式追到社会关系。
3. **Production–accumulation–reproduction loops** — follow how systems reproduce labor, ownership, and legitimacy.  
   **生产—增殖—再生产回路**——追踪系统如何重造劳动力、所有权与合法性。
4. **Double production of contradiction** — see how one mechanism creates both capacity and domination.  
   **内在矛盾的双重生产**——理解同一机制如何同时制造能力与支配。
5. **Political mediation** — structural positions become forces through organization, alliances, states, and ideology.  
   **政治中介**——结构位置要经过组织、联盟、国家与意识形态才会成为现实力量。
6. **Practice and path conditions** — keep the direction, but do not copy a historical route as a universal recipe.  
   **实践与路径条件**——方向可以坚定，路线不能复制成普遍食谱。

## Use it · 使用方法

Install the repository as a Codex skill, then invoke it with prompts such as:

```text
Use Marx’s perspective to analyze why this platform lowers worker income.
用马克思视角分析这家平台为什么压低劳动者收入。
```

```text
Use historical materialism to examine the housing crisis. Do not role-play; use only the framework.
用历史唯物主义分析住房危机。不要角色扮演，只使用框架。
```

```text
Analyze generative AI through ownership, labor, reproduction, and contradiction.
从所有权、劳动、再生产与矛盾的角度分析生成式 AI。
```

Common activation phrases include:

- “用马克思视角 / 用马克思主义分析……”
- “从历史唯物主义、阶级、剩余价值或商品拜物教看……”
- “马克思会怎么看……”
- “Marx perspective”

## Research and boundaries · 研究与边界

- The skill is based on primary texts, correspondence, speeches, historical records, scholarly reviews, and version-critical materials.
- It distinguishes **verified facts**, **textual evidence**, **framework-based inference**, and **unknowns**.
- Contemporary factual questions should be researched before answering; source conflicts and dates must remain visible.
- It does not treat Marx’s historical trends as short-term predictions.
- *Capital* Volumes II and III were edited by Engels from unfinished manuscripts; co-authored texts, drafts, interviews, and later interpretations are labeled accordingly.
- It does not reproduce racist, antisemitic, sexist, dehumanizing, or privately abusive language.
- It is not investment, legal, medical, immigration, or personal-safety advice.
